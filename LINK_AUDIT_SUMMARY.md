## Replacement Strategy

When replacing broken links:

1. **Prefer:** Original sources > Academic sources > Aggregators
2. **Check:** Is the content still relevant to the tooltip/context?
3. **Update:** Both URL and data-tooltip if meaning changed
4. **Document:** Why you chose the replacement (in commit message)

Example:
```tsx
// OLD (broken)
<a href="https://themarginalian.org/xyz" data-tooltip="Article about X">X</a>

// BETTER (working primary source)
<a href="https://original-author.com/work" data-tooltip="Author's original essay on X">X</a>
```
